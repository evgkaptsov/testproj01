Project: **State complexity, two-way NFAs, and graph languages**

This project focuses on special types of finite automata, called 2NFAs and 2DFAs (two-way NFAs and DFAs), which occupy an intermediate position between ordinary NFAs and Turing machines. A connection between these automata and directed graphs was established as early as the late 1970s, but many related questions still remain open. There are interesting results concerning how the number of states of an automaton grows when a 2NFA is converted into an equivalent 2DFA. These results are also known to be related to the P vs. NP problem. In this project, we aim to study existing findings and possibly formulate new statements. We also plan to develop a program that simplifies the conversion of 2NFAs into 2DFAs and helps estimate the resulting state complexity.
